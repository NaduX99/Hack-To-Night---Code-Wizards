Nova Bank Project Report

Project: Hack-To-Night 2026 Challenge / Nova Bank
Stack: Next.js 16, React 19, Tailwind CSS, MySQL, Firebase, Nodemailer, Biome
Report output: reports/project-report.html and reports/project-report.pdf

Summary: This report documents the project functions, new features added, errors found during development, how those errors were solved, and the tools/methods used to verify the application.

1. Project Overview

Nova Bank is a digital banking web application with public landing pages, account authentication, customer dashboard features, bank transfer flows, bill payments, savings goals, e-statements, profile/settings management, notifications, and an AI banking assistant.

5. Errors Found and Solutions

Error / IssueCauseSolution
Standalone carosel folder was not integrated It contained plain HTML/CSS and referenced missing assets/* images with unrelated Marvel content. Recreated the carousel as a React component using existing banking images from public/.
Missing protected routes for new pages /settings and /notifications were not listed in the route guard. Added both to protectedPages and middleware matcher in proxy.ts.
Notifications UI missing The API existed but no user-facing page was present. Created /notifications page with refresh, empty, loading, and error handling.
Settings UI missing Profile page existed, but there was no dedicated settings page. Created /settings page and reused /api/profile for data updates.
Biome formatting/import errors Manual file edits created formatting and import-order differences. Ran npx biome format --write on touched files and verified with npm run lint.
UTF-8/mojibake text such as Â© PowerShell write encoding produced incorrect display for special characters. Rewrote affected files with UTF-8 without BOM and changed footer copy to ASCII-safe text.
TypeScript error: missing CSSProperties Carousel inline custom CSS variables needed a type import. Added import type { CSSProperties } from 'react'.
npm not valid Win32 app when starting server Windows requires the command shim for background launch. Used npm.cmd for Start-Process.
spawn EPERM during build/dev in sandbox Windows sandbox blocked process spawning for Next workers. Reran build/dev with scoped elevated approval. Build completed successfully.
Landing hero looked too plain/centered Original hero was mostly text-centered with limited visual hierarchy. Rebuilt hero with asymmetric layout, animated finance cards, scan line, grid, and gradient headline.
